## Getting Started

To begin using Vincent v.B. for your documentation:

1. Sign in at `https://vincentvb.com`
2. Click "Create Project" on your dashboard
3. Add a project title and description
4. Start adding documentation pages
5. Use the sidebar to organize sections

Docs auto-save as you work. Access all projects anytime from your dashboard.
