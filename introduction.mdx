## Overview

Vincent v.B. is a comprehensive documentation management platform that empowers you to create, organize, and share project documentation with ease. You can build structured docs sites, collaborate with teams in real-time, and publish polished documentation without needing advanced technical skills.

This platform suits developers, product managers, and technical writers who need a centralized space for API references, user guides, and changelogs. Whether you manage open-source projects or internal knowledge bases, Vincent v.B. streamlines your workflow and ensures your documentation stays up-to-date and accessible.

## Quick Start

Get up and running in minutes with these essential steps.

<Steps>
  <Step title="Sign Up" icon="user-plus">

    Visit `https://app.vincentvb.com` and create a free account using your email or GitHub.

  </Step>
  <Step title="Create Project" icon="plus">

    Click **New Project** and select a template for API docs, user guides, or changelog.

  </Step>
  <Step title="Add Content" icon="edit-3">

    Use the editor to write your first page. Start with Markdown:

````markdown
# Welcome

This is your first document.
````

  </Step>
  <Step title="Publish" icon="globe">

    Preview changes, then click **Publish** to deploy your site.

  </Step>
</Steps>

<CodeGroup tabs="CLI,REST API">
````bash
# Install CLI
npm install -g @vincentvb/cli

# Initialize project
vincent init my-docs
````
````javascript
// API call to create project
fetch('https://api.vincentvb.com/v1/projects', {
  method: 'POST',
  headers: { 'Authorization': `Bearer ${YOUR_API_KEY}` },
  body: JSON.stringify({ name: 'My Docs' })
});
````
</CodeGroup>
