## Upgrade Instructions

Follow these steps to upgrade Vincent v.B. to the latest version. Always back up your data first.

<Steps>
  <Step title="Backup Data" icon="database">

  Export your projects:

  ````bash
  vincent export --all --output ./backup
  ````

  </Step>
  <Step title="Update Package" icon="download">

  <CodeGroup tabs="npm,yarn">
  ````bash
  npm install vincent@latest
  ````
  ````bash
  yarn add vincent@latest
  ````
  </CodeGroup>

  </Step>
  <Step title="Run Migration" icon="settings">

  ````bash
  vincent migrate --version 1.2.0
  ````

  Verify with `vincent --version`.

  </Step>
  <Step title="Restart Service" icon="refresh-cw">

  Restart your Vincent instance and test key features.

  </Step>
</Steps>

<Callout kind="tip">
  Test upgrades in a staging environment first to avoid disruptions.
</Callout>
